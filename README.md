# 批量保存vx_wechat朋友の圈图片

## 步骤
1. 先在手机上刷朋友圈，要一张张图打开一下才会缓存高清图片。
2. 将手机连接电脑，微信朋友圈图片缓存通常存储在以下位置：/storage/emulated/0/Android/data/com.tencent.mm/cache/
   在这个路径下，你会看到一个或多个由随机数字和字母组成的文件夹（即 [hash值]），每个对应一个你登录过的微信账号。你需要点进去，找到名为 sns 的文件夹，里面存放的就是你刷过的朋友圈缓存 。图片文件通常以 snsb_ 开头。
4. 将路径下的文件夹里所有文件导出到电脑
5. 将那些文件与该Python程序放在一个目录里，运行Python文件
6. 程序会遍历目录下所有文件，为每个文件添加 .jpg 后缀
7. 可以通过文件名开头筛选出高清图片snshd
